# PixelPast – Vision

PixelPast is a local-first, sovereign, searchable and visually explorable
life data platform.

It aggregates heterogeneous personal data sources (photos, calendars, music,
finance, location history, trips, work activity, sports, etc.) into a unified,
time-based model and renders them as an interactive, multi-year timeline.

---

## North Star

PixelPast transforms fragmented digital traces into a coherent,
queryable and visually understandable life chronology.

Every design and implementation decision must pass these questions:

1. Does it improve temporal exploration?
2. Does it strengthen cross-source linking?
3. Does it preserve data ownership and reproducibility?
4. Does it keep the system maintainable and extensible long-term?

---

## Core Principles

- Local-first
- Full data ownership
- Reproducible ingestion
- Idempotent processing
- Modular source connectors
- Exploration over CRUD
- Visualization as a first-class feature

---

## Product Goal

A zoomable, multi-year calendar heatmap (GitHub-inspired)
that represents each day as an aggregated activity pixel
and allows drill-down into a detailed day-level storyline.
